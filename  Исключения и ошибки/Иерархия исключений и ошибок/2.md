# Практика #2

Ниже маленькие примеры: сначала ошибка, потом обработка через `try/catch`.

## ArithmeticException (деление на 0)

```java
try {
    int a = 5 / 0;
} catch (ArithmeticException e) {
    System.out.println("Деление на ноль");
}
```

## ArrayIndexOutOfBoundsException (индекс вне массива)

```java
try {
    int[] arr = {1, 2, 3};
    System.out.println(arr[10]);
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Неверный индекс");
}
```

## IllegalArgumentException (неверный аргумент)

```java
try {
    int age = -1;
    if (age < 0) throw new IllegalArgumentException("age < 0");
} catch (IllegalArgumentException e) {
    System.out.println("Неправильный аргумент: " + e.getMessage());
}
```

## ClassCastException (не то приведение)

```java
try {
    Object x = "text";
    Integer n = (Integer) x;
} catch (ClassCastException e) {
    System.out.println("Нельзя так привести тип");
}
```

## NullPointerException (обращение к null)

```java
try {
    String s = null;
    System.out.println(s.length());
} catch (NullPointerException e) {
    System.out.println("Ссылка была null");
}
```
