# Практика #3

```java
protected interface Interface { ... }
```

Если это интерфейс верхнего уровня (top-level), так объявить нельзя — ошибка компиляции (для top-level допустимы только `public` или отсутствие модификатора).
`protected` возможен только у вложенного (member) интерфейса — тогда он доступен в своём пакете и в классах-наследниках.
