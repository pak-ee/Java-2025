Как запускать (если нужно):

1) Открыть папку java_practices
2) Скомпилировать файл, например:
   javac Practice1Primitives.java
3) Запустить:
   java Practice1Primitives

Файлы:
- Practice1Primitives.java — все базовые типы + "нулевые" значения
- Practice2UnicodeName.java — имя через \uXXXX
- Practice3Scope.java — область видимости (там показано, почему b не видно)
- Practice4StringPlus.java — примеры String + (и ловушка 10 + 5)
- Practice5MixedArithmetic.java — смешанные типы в арифметике
- Practice6NarrowingCast.java — сужающие преобразования
- Practice7ByteCast.java — почему byte = a + ... не компилируется + правильный вариант
- Practice8InterfaceVar.java — interface и var
