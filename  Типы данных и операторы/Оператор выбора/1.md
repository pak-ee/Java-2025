Практика #1  
Допустимые типы в `switch`: примитивы `byte`, `short`, `char`, `int` (и их обёртки `Byte`, `Short`, `Character`, `Integer`), перечисления `enum`, строки `String` (с Java 7). `long/float/double/boolean` нельзя.

Примеры:
- Примитив / обёртка:  
```java
Byte code = 2;
switch (code) {
    case 1 -> System.out.println("one");
    case 2 -> System.out.println("two");
    default -> System.out.println("other");
}
```
- Символ:  
```java
char ch = 'A';
switch (ch) {
    case 'A' -> System.out.println("first");
    case 'B' -> System.out.println("second");
    default -> System.out.println("other");
}
```
- Перечисление:  
```java
enum Day { MON, TUE, WED }
Day day = Day.TUE;
switch (day) {
    case MON -> System.out.println("monday");
    case TUE -> System.out.println("tuesday");
    default -> System.out.println("other");
}
```
- Строка:  
```java
String cmd = "start";
switch (cmd) {
    case "start" -> System.out.println("go");
    case "stop" -> System.out.println("stop");
    default -> System.out.println("unknown");
}
```
