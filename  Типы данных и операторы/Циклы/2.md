# Практика 2

Пример `break` и `continue` в классическом цикле `for`:

```java
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {      // пропускаем чётные числа
        continue;
    }
    if (i > 7) {           // выходим, когда i превышает 7
        break;
    }
    System.out.println(i); // вывод: 1, 3, 5, 7
}
```
