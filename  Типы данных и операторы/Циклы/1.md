# Практика 1

Два тривиальных примера бесконечных циклов.

```java
int counter = 0;
while (true) {
    counter++;
}
```

`while` не завершится, потому что условие всегда `true`.

```java
int counter = 0;
do {
    counter++;
} while (true);
```

`do-while` тоже не закончится: проверка после тела всегда даёт `true`.
