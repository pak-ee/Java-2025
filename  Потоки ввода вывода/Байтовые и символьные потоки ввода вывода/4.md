# Практика 4

`AutoCloseable` нужен, чтобы объект можно было автоматически закрывать в `try-with-resources`: `close()` вызовется гарантированно, даже если внутри блока произошло исключение. Это предотвращает утечки ресурсов (файлы, сокеты, потоки, соединения с БД).

Пример (встроенный ресурс — `FileInputStream`):

```java
import java.io.FileInputStream;
import java.io.IOException;

public class DemoAutoCloseable {
    public static void main(String[] args) throws IOException {
        try (FileInputStream in = new FileInputStream("data.bin")) {
            int firstByte = in.read();
            System.out.println(firstByte);
        } // in.close() вызовется автоматически
    }
}
```
