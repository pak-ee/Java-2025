# Практика 3

`InputStream`/`OutputStream` работают с *байтами* и не знают про символы и кодировки. `Reader`/`Writer` нужны, чтобы:

- работать с *символами* (`char`/`String`), а не с сырыми байтами;
- корректно преобразовывать байты ↔ символы с учётом *кодировки* (например, UTF‑8), через `InputStreamReader`/`OutputStreamWriter`;
- упростить текстовые операции: `BufferedReader.readLine()`, `PrintWriter.println()` и т.п.;
- избежать ошибок вида “прочитал по байту — сломал многобайтный символ”.
