# Практика 1

Пример с `ByteArrayInputStream` (подкласс `InputStream`), демонстрация `read()`:

```java
import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.nio.charset.StandardCharsets;

public class DemoRead {
    public static void main(String[] args) throws IOException {
        byte[] data = "Hi".getBytes(StandardCharsets.UTF_8);

        try (ByteArrayInputStream in = new ByteArrayInputStream(data)) {
            int b1 = in.read(); // читает 1 байт (0..255) или -1, если конец потока
            int b2 = in.read();
            int b3 = in.read(); // здесь будет -1

            System.out.println(b1);        // 72 ('H')
            System.out.println((char) b2); // 'i'
            System.out.println(b3);        // -1
        }
    }
}
```
