# Практика #1 — `java.util.Scanner` (чтение из файла)

Пример кода для чтения данных из текстового файла с помощью `Scanner`.

## Файлы

- `Practice1.java` — читает файл построчно (через `hasNextLine()` / `nextLine()`).
- `input.txt` — пример входного файла.

## Запуск

Нужен установленный JDK (команды `javac` и `java`).

Скомпилировать:

```bash
javac Practice1.java
```

Запустить (прочитает `input.txt` из текущей папки):

```bash
java Practice1
```

Или указать свой файл первым аргументом:

```bash
java Practice1 path/to/file.txt
```
