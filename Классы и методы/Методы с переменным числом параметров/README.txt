Как запустить:

1) Открой терминал в папке practice_varargs
2) Скомпилируй:
   javac Main.java
3) Запусти:
   java Main

Что смотреть в коде:
- Класс VarargLogger: два метода log(...), оба с varargs, но разные параметры (перегрузка).
- В main вызываются оба варианта, чтобы увидеть разницу в выводе.
