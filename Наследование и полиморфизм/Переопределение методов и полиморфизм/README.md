# Практика: overloading / overriding / @Override

## Требования
- Установлен JDK (Java Development Kit) и доступны команды `java` и `javac`.

Проверка:
```bash
java -version
javac -version
```

## Сборка (компиляция)
Из папки с файлами `.java`:
```bash
javac *.java
```

## Запуск
Практика #1 (перегрузка методов):
```bash
java Practice1Overloading
```

Практика #2 (переопределение + тип возвращаемого значения):
```bash
java Practice2Overriding
```

Практика #3 (как @Override помогает найти ошибку):
```bash
java Practice3OverrideAnnotation
```
