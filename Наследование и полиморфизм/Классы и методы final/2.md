## Практика №2 — final-класс нельзя наследовать

Если класс объявлен как `final`, то от него **нельзя** создать подкласс — тоже будет ошибка компиляции.

```java
final class Animal {
    public void voice() {
        System.out.println("Какой-то звук");
    }
}

class Dog extends Animal { // ОШИБКА: нельзя наследоваться от final-класса
}
```

Что примерно скажет компилятор:
`cannot inherit from final Animal`
