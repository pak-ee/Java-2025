## Зачем нужны три двойные кавычки (`"""`) в Java?

Три двойные кавычки используются для текстовых блоков (Text Blocks) — это многострочные строковые литералы.

Что дают текстовые блоки:
удобно писать строки в несколько строк (SQL, JSON, HTML и т.п.);
меньше экранирования: не нужно ставить `\n` и постоянно экранировать кавычки;
сохраняется форматирование (переводы строк и отступы контролируются правилами text block).

Пример:


String json = """
    {
      "name": "Alex",
      "age": 20
    }
    """;


Текстовые блоки добавлены в Java 15 (как финальная возможность; раньше были в preview).
