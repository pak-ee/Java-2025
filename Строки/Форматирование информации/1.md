# Практика №1 — примеры спецификаторов `java.util.Formatter`

Пример (5 спецификаторов: `%b`, `%c`, `%d`, `%f`, `%s`):

```java
import java.util.Formatter;

public class Main {
  public static void main(String[] args) {
    boolean ok = true;
    char grade = 'A';
    int count = 42;
    double pi = Math.PI;
    String name = "Ирина";

    try (Formatter f = new Formatter()) {
      f.format("Логическое: %b%n", ok);     // %b
      f.format("Символ: %c%n", grade);      // %c
      f.format("Целое: %d%n", count);       // %d
      f.format("Дробное: %.3f%n", pi);      // %f
      f.format("Строка: %s%n", name);       // %s
      System.out.print(f.toString());
    }
  }
}
```
