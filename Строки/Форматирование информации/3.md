# Практика №3 — примеры спецификаторов даты/времени `java.util.Formatter`

Пример (5 спецификаторов: `%tH`, `%tM`, `%tS`, `%tY`, `%tB`):

```java
import java.util.Date;
import java.util.Formatter;

public class Main {
  public static void main(String[] args) {
    Date now = new Date();

    try (Formatter f = new Formatter()) {
      f.format("Час (00–23): %tH%n", now);      // %tH
      f.format("Минуты: %tM%n", now);           // %tM
      f.format("Секунды: %tS%n", now);          // %tS
      f.format("Год (4 цифры): %tY%n", now);    // %tY
      f.format("Месяц (полностью): %tB%n", now);// %tB (зависит от Locale)
      System.out.print(f.toString());
    }
  }
}
```
